# Landing Page Notes

- This scope covers only routed page-layer work for the public landing site:
  - `/` landing page
  - `/support/:language` user support page
  - `/privacy/:language` privacy page
- Keep edits focused on page copy, page composition, page-level styling, and existing page assets/components.
- Use `src/App.tsx` to confirm routing context before changing routed pages.
- Use the Figma reference design document for landing page visual direction: [Kira Snap Landing Scrapbook](https://www.figma.com/design/RTktthjWooXlTIsnKpUzCX/Kira-Snap-Landing-Scrapbook?node-id=0-1&p=f&t=dgDSGpknjnnsBQdD-0).
- Treat the Figma file as a reference design document, not an automatic sync source.
- Do not use this scope for backend, iOS, admin UI, deployment, build tooling, or non-page infrastructure work.
